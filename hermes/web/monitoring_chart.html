Lesson context

Task

Activity 1 — Student Task Statement
1. How are the tools alike? How are they different? 2. Find the number that makes the equation true. Show your thinking using drawings, numbers, or words.curriculum/im_teacher_guides/grade2/unit2/lesson2.md, line 180

Goal

Find unknown addends by adding or subtracting within 100, and explain how the representation or place-value strategy works.

Standards

  • IM-G2-U2-L2 im_lesson
    IM Grade2 Unit 2 Lesson 2: Find the Unknown Addend. The purpose of this lesson is for students to find the value of unknown addends by adding or subtracting within 100. Goals: Explain (orally) representations and strategies used to find an unknown addend, including those based on place value.
  • 2.NBT.A.2 ccss
    Count within 1000; skip-count by 5s, 10s, and 100s.
  • 2.NBT.B.5 ccss
    Fluently add and subtract within 100 using strategies based on place value, properties of operations, and/or the relationship between addition and subtraction.
  • 2.CA.2 in_indiana
    Using number sense and place value strategies, add and subtract within 1,000, including composing and decomposing tens and hundreds. Use models, drawings, and strategies based on place value, properties of operations, and/or the relationship between addition and subtraction; describe the strategy and explain the reasoning used.
  • 2.NBT.B.9 ccss
    Explain why addition and subtraction strategies work, using place value and the properties of operations.

What to monitor

  • Count on from the larger numberListen for a clear starting number, counted-on addend, successive counts, and a final total.Ask: “How did your starting number help you find the unknown?”
  • Combine or separate tens and onesListen for tens and ones kept aligned, with the unknown represented as a missing part.Ask: “Where is the unknown in your blocks or drawing?”
  • Make a ten and preserve the leftoverListen for a split addend, the part needed to make ten, and the leftover conserved in the result.Ask: “What happened to the part you did not use to make ten?”

Discussion order

Activity 1 — Activity Synthesis
• Invite previously selected students to share in the given order. Record or display their work for all to see. • Connect students’ approaches by asking: ◦ “How are these ways for making the equation true the same? How are they different?” (Some people did the same steps, but they used different tools to show it. Some ways used the same tool, but they are different because one group added tens and ones to find the unknown number, and another group took away tens and ones to find the unknown number.) • Connect students’ approaches to the learning goal by asking: ◦ “What did you notice about the different tools we used?” ◦ “What is different about using connecting cubes and using base-ten blocks?” ◦ “What is different about using a diagram instead of using blocks?”curriculum/im_teacher_guides/grade2/unit2/lesson2.md, line 225

Assess: What quantity is unknown? How does the representation show it?

Advance: How could you use a different tool to find the same unknown?
