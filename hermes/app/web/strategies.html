Strategies, running

Children do not compute the way calculators do. A child finding 38 + 27 might count on by tens and then by ones, or chunk the 27 apart, or rearrange to make a friendly base first. Researchers in the Cognitively Guided Instruction tradition spent decades documenting these strategies; this system encodes each one as a small machine — a finite-state automaton whose every step is inspectable.

What you run below is not an animation written for this page. The page sends your numbers to the actual machine, and the machine sends back its execution history — each state it passed through, and the number-line jumps it took. Pick a strategy, change the numbers, watch how differently each machine moves.
