Where the table becomes ours. Until now each fold, definition, and teacher sentence has been yours alone. This prompt is the moment where individual findings start to consolidate into a shared object the class makes together — a properties table that survives any one student stepping away from it.

We have been working on activities to make sense of quadrilateral properties and you have been compiling a table of quadrilateral properties. We will now summarize our findings in two steps. First you will share here 2 properties you have found for one or more quadrilaterals. Reply to the thread that corresponds to the quadrilateral you are sharing properties about, and make it clear what kind of property you are sharing (sides, angles, diagonals, or symmetry). Share properties that have not been shared already. After sharing in this discussion, add your properties to the class table on the next page.

Purpose

To put two of your own findings into a shared artifact the whole class will use, and to defend each finding against the kind of pressure a classmate could realistically apply.

Initial Post

Before reading classmates’ posts on this thread, make a property-evidence artifact.

  1. Name the quadrilateral (or quadrilaterals) you are contributing to.
  2. State two properties not already shared in this thread.
  3. For each, name the kind: sides, angles, diagonals, or symmetry.
  4. For each, give one sentence naming the evidence that the property survives: a fold, a construction, a counterexample it rules out, or a definitional consequence.
  5. Name one property you considered but decided not to claim, and say why you held it back.
  6. Write one teacher sentence you could say to a student who asserts the same property without evidence.

End with this sentence:

I would defend these properties only if…

Aim for 200–300 words.

Peer Uptake Reply

Reply to a classmate who is contributing to the same quadrilateral thread as you, or to a classmate who claimed the same kind of property (sides / angles / diagonals / symmetry) for a different quadrilateral. Use their claim. In your reply:

  1. Name one part of their evidence you would inherit into your own table entry.
  2. Name one missing test or hidden assumption their property depends on.
  3. Offer one fold, counterexample, or special case that would put pressure on the claim.
  4. Add one sentence beginning:

I would not add this property to the class table yet if…

Return Post

Return to your initial post after the thread has more contributions. Revise one of your properties, defend it, or name the exact verification still needed before you would add it to the class table.

End with this sentence:

My property became part of the class’s table when…

If no one has replied to you by the return-post window, use the reply you wrote to a classmate as your outside pressure and say so.

Requirements

Make your initial post, peer uptake reply, and return post by the dates listed in Canvas. After your return post, add your accepted properties to the class table on the next page.

Where this points: The table speaks back. Once the bullets are added, the table is no longer a sum of contributions — it is a public object that pressures everyone’s definitions at once.
