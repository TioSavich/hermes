CONTRIBUTE    

The table speaks back. A list of properties is not yet a table. A table is a public object: it surfaces what holds across categories, what conflicts, what is still missing. Adding your bullets is only the first move. Reading the table as something the class made together is the harder one.

Purpose

To consolidate the class’s findings into a shared table, then read the table back as a public object that pressures everyone’s definitions at once.

Parallelogram

A quadrilateral with both pairs of opposite sides parallel. 

Side Properties
  • Opposite sides are equal in length
  • Both pairs of opposite sides are parallel
Angle Properties
  • Opposite angles are equal
  • Adjacent angles are supplementary 
Diagonal Properties
  • Diagonals bisect each other
  • Diagonals are not necessarily equal in length
Symmetry Properties
  • Rotational symmetry of 180° about where the diagonals intersect
  • Does not have line symmetry

Read the Table Back: Initial Post

After your bullets are entered, read the table as a finished object. Make a short reflection artifact and post it as a discussion reply to this page (or to the parallel SHARE thread if your instructor sets one up).

  1. Name one row where the bullets agree across categories (e.g., parallelogram side properties echo rectangle side properties). What invariant is the table showing you?
  2. Name one row where the bullets conflict or pull in different directions. What would have to be decided to resolve the conflict?
  3. Name one cell that is still empty or thin. What property would you guess goes there, and what would you check first?
  4. Name one bullet whose initials are not yours that you trust, and one whose claim you would verify before passing on.
  5. Say what the table now does that no single one of you could have done alone.

End with this sentence:

The table tells me that…

Aim for 200–300 words.

Peer Uptake Reply

Reply to a classmate’s reflection. In your reply:

  1. Name one of their observations about the table that you would inherit.
  2. Name one place where you read the same row differently, and say what you notice that they did not.
  3. Offer one test, definition, or counterexample that would help the class settle the disagreement.
  4. Add one sentence beginning:

I would not trust this row of the table yet if…

Return Post

Return to your own contribution (the bullet with your initials). After classmates have added more, revise the bullet, defend it, or strengthen it with the test or counterexample that now makes it survive.

End with this sentence:

The table changed my own bullet when…

Share 2–3 bullets with quadrilateral properties in the related discussion post, then add them to this table with your initials. Post your reflection, peer reply, and return post by the dates listed in Canvas.
