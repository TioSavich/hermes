# Count Columns and Objects in Columns
_Anchor ID: `IM-G2-U8-L8` — Grade grade2, Unit 8, Lesson 8_

## Learning Goals (teacher-facing)
- Comprehend (in spoken and written language) the term “column” refers to objects arranged up and down in an array.
- Create arrays and describe (orally) the “rows” and “columns” of arrays.

## Student-Facing Learning Goals
Let’s learn about columns in arrays.

## Lesson Purpose
The purpose of this lesson is for students to see that in an array of objects, the objects are arranged into columns with an equal number of objects in each column.

Lesson Purpose
The purpose of this lesson is for students to see that in an array of objects, the objects are arranged into columns with
an equal number of objects in each column.


Narrative
In the previous lesson, students learned that an array is an arrangement of objects into rows with an equal number of
objects in each row.

In this lesson, students refine their understanding of an array to include the fact that the objects in each row start in the
same place and are equally spaced from each other. When arranged this way, the objects also line up into columns.
Students use mathematical language to describe arrays and recognize that rows go side to side and columns go up and
down (MP6). They use the structure of the array to find the total number of objects in an array (MP7).

Students should have access to counters throughout the lesson, including during the Cool-down.

Count by Columns

     Standards                                                       Instructional Routines
Addressing      2.OA.C.4                                          • MLR8 Discussion Supports


The purpose of this activity is for students to describe the structure of an array by identifying the number of columns,
the number of objects in each column, and the total number of objects. Students learn that the columns of an array go
up and down. Students begin to reason about how they can use the structure of an array to find the total number of
objects without counting by 1.

This activity uses MLR8 Discussion Supports. Advances: speaking, conversing.


      Access for Students with Disabilities
      Representation: Develop Language and Symbols. Support understanding of the rows and columns by inviting
      students to act it out. For example, use students to create the first array (3 rows of 2). Discuss the students who
      create the rows. Now switch to talking about columns (2 columns of 3). Discuss the difference between how
      students create the rows and how they create the columns.
      Supports accessibility for: Visual-Spatial Processing, Organization

in the other rows. The counters in an array also line up in columns that go up and down. The columns in an array always
have the same number of counters in them. Arrays can help us organize objects, so we can find total amounts.”
