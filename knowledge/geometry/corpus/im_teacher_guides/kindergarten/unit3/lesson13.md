# Describe and Match Shapes
_Anchor ID: `IM-GK-U3-L13` — Grade kindergarten, Unit 3, Lesson 13_

## Learning Goals (teacher-facing)
- Comprehend (in spoken language) the terms “above,” “below,” “next to,” and “behind” are used to describe the location of objects.
- Describe (orally) the location of shapes, using “above,” “below,” “next to,” and “behind.”

## Student-Facing Learning Goals
Let’s build matching shapes.

## Lesson Purpose
The purpose of this lesson is for students to use positional words to describe the location of pattern blocks in a larger shape.

Introduce Match Mine—Pattern Blocks

     Standards
Addressing     K.G.A.1


The purpose of this activity is for students to learn Stage 1 of the Match Mine center. One student makes an object,
using pattern blocks and without letting their partner see. They use positional words to describe the location of pattern
blocks within the object, and then their partner tries to make the same object based on the description. Students may
initially describe the objects generally or use positional words other than those introduced in the previous activity.
Students need repeated experiences hearing positional words in order to develop precision with their language (MP6).

Folders or other objects can be used to keep students’ work hidden.


      Access for English Language Learners
      MLR8 Discussion Supports. For each description that is shared, invite students to turn to a partner and restate
      what they heard, using precise mathematical language, specifically positional words.
      Advances: Listening, Speaking
