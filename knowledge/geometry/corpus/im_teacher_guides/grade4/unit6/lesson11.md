# Partial Products and the Standard Algorithm
_Anchor ID: `IM-G4-U6-L11` — Grade grade4, Unit 6, Lesson 11_

## Learning Goals (teacher-facing)
- Calculate the product of three- or four-digit numbers and one-digit numbers.
- Compare and contrast (orally and in writing) algorithms that use partial-products, including the standard algorithm for multiplication

## Student-Facing Learning Goals
Let’s compare multiplication algorithms.

## Lesson Purpose
The purpose of this lesson is for students to analyze the standard algorithm for multiplication and compare it to an algorithm that uses partial products they saw in earlier lessons.

Lesson Purpose
The purpose of this lesson is for students to analyze the standard algorithm for multiplication and compare it to an
algorithm that uses partial products they saw in earlier lessons.


Narrative
In previous lessons, students analyzed and used an algorithm that uses partial products to multiply multi-digit whole
numbers. They learned that an algorithm can represent the base-ten diagrams and rectangular diagrams, but it is more
efficient for keeping track of and recording partial products.

This lesson extends students’ analysis to include the standard algorithm for multiplication of multi-digit numbers. In IM
Grade 4, the standards focus on understanding place value and how it is represented in different methods for finding
products. The work here serves to build the groundwork for making sense of the standard algorithm in IM Grade 5, so
students are not expected to use the standard algorithm at this time.
