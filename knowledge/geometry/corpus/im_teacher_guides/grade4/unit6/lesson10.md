# Using Algorithms with Partial Products: 2 Two-Digit Numbers
_Anchor ID: `IM-G4-U6-L10` — Grade grade4, Unit 6, Lesson 10_

## Learning Goals (teacher-facing)
- Interpret (orally and in writing) an algorithm that uses partial products to multiply 2 two-digit numbers.
- Use an algorithm that uses partial products to multiply 2 two-digit numbers.

## Student-Facing Learning Goals
Let’s try to multiply two-digit numbers with an algorithm that uses partial products.

## Lesson Purpose
The purpose of this lesson is for students to use partial products in an algorithm to multiply 2 two-digit numbers.

Prior to this lesson, students have not needed to compose units when finding the sum of partial products. In the Warm-
