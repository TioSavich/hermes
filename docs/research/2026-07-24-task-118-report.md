`IM-GK-U1-L3`, with one worker so the interruption occurred between completed
lessons.

2. It received `SIGINT`, preserved that partial, and exited with status 130.
3. The resumed builder reported `resumed=1 pending=2` and checkpointed the

4. Two merges from the resumed partial set were byte-identical.
5. A separate uninterrupted three-lesson build merged to the same bytes.
