## Result

Representation rendering now requires the mention's stated partition and unit
to join an action record in its utterance or extraction window. A drawable
record contains a `binding` receipt with the matched action ids, utterances,
and the exact quantities each action supplied. The renderer uses only drawer
seams that express that extent: jumped number lines and grids of partitioned
rectangles. Other kinds abstain rather than receiving a nearby scene.
