# Build Shapes
_Anchor ID: `IM-GK-U7-L1` — Grade kindergarten, Unit 7, Lesson 1_

## Learning Goals (teacher-facing)
- Determine the number of pattern blocks used to compose a design, and represent the quantities with written numbers.

## Student-Facing Learning Goals
Let’s use one shape.

## Lesson Purpose
The purpose of this lesson is for students to reinforce counting concepts as they build with and consider the attributes of pattern blocks.

Lesson Purpose
The purpose of this lesson is for students to reinforce counting concepts as they build with and consider the attributes
of pattern blocks.


Narrative
In previous units, students put together pattern blocks to form larger shapes and filled in puzzles. They counted groups
of up to 20 objects and images and wrote numbers to record their count. Students use only one kind of pattern block to
fill in puzzles and eventually create given shapes without using outlines, which requires students to think informally
about the attributes of shapes. Students need to change the orientation of the pattern blocks and align the sides of the
pattern blocks. Students may be able to visualize how to turn or flip the shape to fill a particular space or may need to
use trial and error. In both activities, students count and write a number to record how many pattern blocks they used.

Squares, Squares, Squares

     Standards
Addressing     K.CC.A.3, K.CC.B.5, K.G.B.6


The purpose of this activity is for students to put together pattern blocks strategically to make a specific shape. Students
use square pattern blocks to fill in two squares. Then they work together to create another square and a shape that is
not a square, without using outlines. Students informally compare the shapes that they have created in the Activity
Synthesis. Students count and write numbers to show how many pattern blocks they used.

Shape Puzzle

     Standards
Addressing     K.CC.A.3, K.CC.B.5, K.G.B.6


The purpose of this activity is to solve a puzzle, using only one pattern block shape. Students need to change the
orientation of the pattern blocks and align the sides of the pattern blocks carefully to fill in the puzzle (MP6). Although
the task asks students to use only one kind of pattern block in the second problem, students may choose to fill in the
puzzle with different kinds of pattern blocks. If they do, they should still write a number to represent how many pattern
blocks they used.


      Access for English Language Learners

      times: Triangle and rhombus.
      Advances: Speaking, Representing


      Access for Students with Disabilities
      Action and Expression: Develop Expression and Communication. Give students access to 10-frames to help them
      count the number of rhombus pattern blocks they used.
