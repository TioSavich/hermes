# Build with Solid Shapes
_Anchor ID: `IM-GK-U7-L14` — Grade kindergarten, Unit 7, Lesson 14_

## Learning Goals (teacher-facing)
- Describe (orally) objects composed of solid shapes, including using positional words.

## Student-Facing Learning Goals
Let’s build with solid shapes.

## Lesson Purpose
The purpose of this lesson is for students to compose solid shapes to build new shapes.

Lesson Purpose
The purpose of this lesson is for students to compose solid shapes to build new shapes.


Narrative
In previous lessons, students described and built solid shapes from components. They used positional words to
describe the relative locations of shapes. In this lesson, students use solid shapes to compose objects. Students may
notice attributes of solid shapes in the context of building, such as that a sphere is round and rolls. The focus is on
listening to the language that students use to describe the shapes that they put together and shapes' relative locations
(MP6).

Build My Shape

     Standards
Addressing      K.G.A.1, K.G.B.6


The purpose of this activity is for students to build objects with solid shapes. Students practice using positional words
and the names of solid shapes as they try to recreate a building (MP6). Initially, students recreate the object that their
partner built. Then students work together to build an object.


      Access for Students with Disabilities

      that they used to build their partner’s design. For example, "I used the cone to create the pointy part of the
      design."
      Supports accessibility for: Memory, Organization

Activity 3                                                                                                     25 min

Introduce Match Mine—Solid Shapes

     Standards
Addressing      K.G.A


The purpose of this activity is for students to learn Stage 2 of the Match Mine center. One student builds an object, using
solid shapes and without letting their partner see. Students then use positional words to describe what they have built,
and their partner tries to build the same object, based on the description. Folders or other objects can be used to keep
students’ work hidden. In order to build matching objects, each partner needs a matching set of solid shapes.

After they participate in the center, students choose any previously introduced stage from these centers:

  • Build Shapes
  • Geoblocks
  • Counting Collections
  • Shake and Spill
