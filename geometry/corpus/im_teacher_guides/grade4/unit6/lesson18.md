# Use an Algorithm with Partial Quotients
_Anchor ID: `IM-G4-U6-L18` — Grade grade4, Unit 6, Lesson 18_

## Learning Goals (teacher-facing)
- Determine (orally and in writing) computation errors when dividing four-digit numbers by one-digit divisors and explain how to correct the errors.
- Use an algorithm with partial quotients to divide up to four-digit numbers by one-digit divisors.

## Student-Facing Learning Goals
Let’s find quotients using an algorithm that uses partial quotients.

## Lesson Purpose
The purpose of this lesson is for students to use an algorithm with partial quotients to find whole number quotients with up to four-digit dividends and one-digit divisors.
