Console

Type a computation and the calculator works it the way a child using a named strategy would, drawing each action on a number line. Discussions

Paste a classroom transcript and the console checks the mathematical claims it can recognize, deterministically: each claim comes back holds, refuted, or not covered, with the worked check attached. The fuller two-pass report asks a language model first, so it needs a key.

Journey index

The More Machine

Feel the snap

A Zeeman catastrophe machine. Drag the blue point and watch continuous tension produce a discontinuous jump. The disc cannot transition smoothly — it must snap. The machine remembers each snap through Cantor's diagonal construction: read the history, negate it, produce something new.

The Bridge

Watch the formalization's internal dynamics

Connects the Prolog meta-interpreter to the catastrophe geometry. Run an arithmetic computation and watch the system's tension — inference count, crisis state, strategy transitions — drive the catastrophe machine in real time. When the learner hits crisis, the machine snaps.

The ORR Explorer

Read the story of learning

The Prolog server's built-in timeline. Shows the narrative of crisis-driven learning as a sequence of events: the attempt, the failure, the teacher consultation, the strategy synthesis, the retry. Each card records what happened at each stage of the ORR cycle — Observe, React, Reorganize.

The Playground

Grab a node, tug the springs

A coupled physics simulation where each state in a strategy automaton is a Zeeman catastrophe machine. Drag control points to build tension. When a node snaps, tension propagates through the network. Enough snaps trigger crisis — and a new strategy ring materializes from the inside out.

Meaning-Use Diagrams

Brandom's MUR over the registry

An interactive Meaning-Use Diagram for the action-automata registry: practices and vocabularies connected by PP-suff, PV-suff, VP-suff, and the LX relation. Defaults to the cross-multiplication neighborhood — the cleanest LX-elaboration the registry encodes — with a picker for the other 27 practices. Grounding metaphors and their break points sit at the bottom.

Representation Visualizations

Draw the mathematical witnesses

Number lines, fraction bars, area models, base-ten blocks, set grouping, and balance-scale filmstrips. Each page uses the worker-shaped render document directly; compare pages put productive schemes beside their deformations.

Unit Echo

Regroup outward, partition inward

Pairs the Ace-of-Base regrouping witness with a Fraction Bars iteration at the same arity. The Prolog bridge retains the opposed action directions and validates a unit-fraction plan reconstructed from the final bar geometry.
