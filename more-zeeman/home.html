Hermes

How children learn arithmetic, made workable.

A child starts with counting by ones. It works until it does not — and the moment it stops working is where a new strategy is built. These pages let you draw that process, read it as a story, and analyze a real classroom discussion against it. Start wherever you like.

The Console is the private teacher tool. It reads student work, so it is not hosted publicly. To run it locally: bash hermes/app/launch.sh, then open http://127.0.0.1:8765/ (this page lives at /more-zeeman/home.html on that same server). The Tools and Journey pages need no setup and make no calls to any language model.
Part of umedcta-formalization, the Prolog and manuscript archive for Understanding Mathematics as an Emancipatory Discipline: A Critical Theory Approach. The model covers whole-number arithmetic through the crisis cycle; for fractions it draws the area model and the rules-without-reason contrast, and it is honest about what it cannot yet model.
