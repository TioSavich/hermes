Does the machine actually reorganize itself?

Don't take the word for it — test it. Type a fraction problem. Watch the machine try everything it knows and get stuck, then build a new method out of two basic actions it already has — cutting a whole into parts, and copying a part — and run that method to get the answer. Change the numbers and it happens again, on a problem it has never seen. Nothing is looked up; nobody hands it the answer.

Being honest about what this is — and isn't

What it does on its own: get stuck (an exhaustive search that provably reaches no answer at its current level), search its own basic actions for a new combination, install that combination, and run it — without being handed the answer and without looking it up.

What "reorganize" does not mean here: the machine is not conscious and wants nothing. The ladder of levels is part of the model we gave it — it comes from how researchers describe the stages children pass through with fractions — not something the machine invented. The honest, interesting line is exactly where it still gets stuck (try an input it can't reach).
