# Make and Break Apart Pattern Block Designs
_Anchor ID: `IM-GK-U5-L2` — Grade kindergarten, Unit 5, Lesson 2_

## Learning Goals (teacher-facing)
- Create designs composed of 2 types of pattern blocks. Describe (orally) this composition using number names for each part and the total.

## Student-Facing Learning Goals
Let’s make and break apart pattern block designs.

## Lesson Purpose
The purpose of this lesson is for students to compose and decompose numbers up to 9 in more than 1 way.
