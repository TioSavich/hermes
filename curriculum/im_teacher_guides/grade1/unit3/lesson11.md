# Add to a Teen Number
_Anchor ID: `IM-G1-U3-L11` — Grade grade1, Unit 3, Lesson 11_

## Learning Goals (teacher-facing)
- Explain (orally) strategies for solving an Add To, Result Unknown problem when one addend is a teen number.
- Find the value of sums within 20 when one addend is a teen number.

## Student-Facing Learning Goals
Let’s add to teen numbers.

## Lesson Purpose
The purpose of this lesson is for students to add within 20 when one addend is a teen number.

Lesson Purpose
The purpose of this lesson is for students to add within 20 when one addend is a teen number.


Narrative
In previous lessons, students learned that a teen number is composed of a ten and some ones and related addition and
subtraction with teen numbers in equations. Students also learned that counting is related to addition, and they added
numbers using the commutative property.

The purpose of this lesson is for students to solve addition story problems within 20 in which one addend is a teen
number. Students then find the unknown value in addition equations where one addend is a teen number. Students
may count all, count on, or recognize that they can add the ones and then add the ten. This lesson prepares students for
future work, when they add within 100 using methods involving place value.

The purpose of this activity is to elicit and discuss methods for adding a one-digit number to a teen number within 20.
Students are presented with a simple story problem type (Add To, Result Unknown) so discussion can focus on the
methods students used to find the sum.

Students represent and solve the problem in a way that makes sense to them (MP1). Some students may build the teen
number, add counters and count all. Other students may count on from the teen number. Some students may see that
the sum will still have 1 ten and just combine the ones. During the Synthesis, students notice that when adding to teen
numbers within 20, the unit of ten in the representation does not change—only the ones change (MP8).
