## Student-Facing Learning Goals
Let’s play some games and practice adding decimals.

## Lesson Purpose
The purpose of this lesson is for students to practice adding decimals.

    hundredths or thousandths.

    sheet. If it is your first turn, add the number

Lesson Synthesis
“Today, we practiced adding decimals. How is adding with decimals the same as adding with whole numbers? How is it
different?” (If I add digits with the same place value, I can use the same method to add decimals as I use to add whole
numbers. I have to make sure to add digits with the same place value so I can't always line up the digits to the right, like I
do with whole numbers.)
