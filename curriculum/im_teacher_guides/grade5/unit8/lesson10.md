# Here Comes the Sum
_Anchor ID: `IM-G5-U8-L10` — Grade grade5, Unit 8, Lesson 10_

## Learning Goals (teacher-facing)
- Add fractions with unlike denominators.

## Student-Facing Learning Goals
Let’s play some games to practice adding fractions.

## Lesson Purpose
The purpose of this lesson is for students to practice adding fractions with unlike denominators.

placed in the numerator in the first game go in the denominator in this game, and similarly the numbers that went in
the denominator in the first game go in the numerator in this game (MP8).
