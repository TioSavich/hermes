# Add and Subtract within 1,000,000
_Anchor ID: `IM-G4-U4-L20` — Grade grade4, Unit 4, Lesson 20_

## Learning Goals (teacher-facing)
- Determine (orally and in writing) errors made when adding or subtracting using the standard algorithm.
- Use the standard algorithm to add and subtract multi-digit numbers with multiple compositions or decompositions.

## Student-Facing Learning Goals
Let’s use the standard algorithm to add and subtract.

## Lesson Purpose
The purpose of this lesson is for students to add and subtract within 1,000,000 with multiple compositions or decompositions.
