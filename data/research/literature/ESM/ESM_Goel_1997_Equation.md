In the July 1995 issue of the Journal of Educational Studies in Mathematics Even et al. (1995) claim that the quantity /n28 /n00 8 /n29 1 /= 3 is undefined. Their reasoning is as follows:

$$- 2 = ( - 8 ) ^ { 1 / 3 } = ( - 8 ) ^ { 2 / 6 } = [ ( - 8 ) ^ { 2 } ] ^ { 1 / 6 } = 2 .$$

Their intent is to show how ill-prepared high school teachers are in Israel. It appears that Even et al. (1995) themselves are not clear about the definition of a r , where r is a rational number.

Theyhavefallen into a common notational trap regarding rational exponents of real numbers. It is precisely because of such pitfalls that care must be taken, both when defining and manipulating such forms.

Current textbooks avoid these difficulties in various ways. Larson et al. (1993, p. 23) define rational exponent as follows:

If ' a ' is a real number and n is a positive integer such that the principal n th rooth of ' a ' exists, then we define a 1 /=n to be

$$a ^ { 1 / n } = \, ^ { n } \sqrt { a } .$$

Moreover, if m is a positive integer that has no common factor with n , then

$$a ^ { m / n } = \, ^ { n } \sqrt { a ^ { m } } .$$

Dugopolski (1995: pp. 28-29) comments that if r /= p/=q is a rational number and the q th root of ' a ' is a real number, then one defines a r by

$$a ^ { r } = a ^ { r _ { 1 } }$$

where r 1 is r reduced to its lowest terms.

Thus, either using the definition of rational exponents given by Larson and Hostetler or using Dugopolski's comment, the quantity /n28 /n00 8 /n29 1 /= 3 is a well-defined real number with value /n00 2 (but not /+ 2).

We continue to study Equation (1), as it is intriguing and demands a closer look. If one follows the erroneous reasoning contained in Equation (1), then all positive real numbers are their own additive inverses. Let ' a ' be any positive real number. By the arguments in Equation (1),

$$- a = ( - a ) ^ { 1 } = ( - a ) ^ { 2 / 2 } = [ ( - a ) ^ { 2 } ] ^ { 1 / 2 } = ( a ^ { 2 } ) ^ { 1 / 2 } = a .$$

Even et al. (1995) claim that /n28 /n00 8 /n29 1 /= 3 is undefined because a definition of a mathematical operation should not depend on its representatives (see Equation (1)). They also claim that vast majority of the teachers surveyed incorrectly argued that /n28 /n00 8 /n29 1 /= 3 /= /n00 2. Further, they stated that the majority of the teachers who argued that /n28 /n00 8 /n29 1 /= 3 /= /n00 2 remained convinced that their definition was correct.

In this paper, we have presented specific reasons why the teachers interviewed in Even et al. (1995) who claimed that /n28 /n00 8 /n29 1 /= 3 /= /n00 2, were indeed correct.
